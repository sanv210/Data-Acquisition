================================================================================
                        TROUBLESHOOTING
================================================================================

Problem: "PyInstaller not found"
Solution: pip install pyinstaller

Problem: "Module not found" error during build
Solution: Run preflight_check.py to see what's missing
          Then: pip install <missing-package>

Problem: Build takes very long
Solution: Normal - first build can take 3-5 minutes
          Subsequent builds are faster

Problem: .exe is very large (100+ MB)
Solution: Expected - includes all Python libraries
          This is normal for PyInstaller executables

Problem: Backend won't start when .exe runs
Solution: 1. Ensure MySQL server is running
          2. Check backend/.env credentials
          3. Check daq_system.log for errors

Problem: "Access denied" when running
Solution: 1. Run .exe as Administrator
          2. Or disable antivirus temporarily
          3. PyInstaller executables sometimes flagged by AV
